## `Result`がすごい!!

エラーは例外で返すものだと思っていた(Python脳)ので、衝撃

例外がなくても案外やっていける

```rust
items.map(|v| may_fail(v))
     .collect::<Result<Vec<_>, _>>()
```

---

## `?`(Try)がすごい!!

[rust-lang-nursery/failure](https://github.com/rust-lang-nursery/failure)と組み合わせると、まるで例外のようにエラー処理がスムーズにできる

```rust
fn may_fail(in: i32) -> Result<i32, Error> {
  let mut buffer = String::new();
  io::stdin().read_to_string(&mut buffer)?;
  let parsed = buffer.parse()?;
  Ok(parsed + in)
}
```

## `Enum`と`match`

```rust
fn eval(expr: Expression) -> i64 {
  match expr {
    Number(n) => n,
    Boolean(tf) => tf as i64,
    Add(lhs, rhs) => eval(*lhs) + eval(*rhs)
  }
}
```
